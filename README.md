한국 부동산(전세·월세·매매)을 찾고 평가한 뒤 중개사에게 연락까지 도와주는 **Claude 스킬(Skill)**입니다.

니즈 인터뷰 → 매물 검색 → 적합도 랭킹 → 전세사기 위험 점검 → 중개사 문의 작성/발송 → 계약 전 실사 핸드오프까지, 집 구하는 전 과정을 단계별로 안내합니다.

집을 처음부터 찾는 흐름뿐 아니라, **이미 봐둔 매물의 위험 점검**이나 **받아본 계약서·특약 검토**처럼 특정 단계만 단독으로 물어봐도 됩니다.

> ⚠️ 본 스킬은 참고용이며 법률·중개 자문이 아닙니다. 계약 전 반드시 등기부등본·건축물대장 확인과 공인중개사·HUG·법률 전문가 상담을 거치세요.

## 주요 기능

- **적응형 니즈 인터뷰** — 거래유형(전세/월세/매매)과 건물유형에 따라 질문을 분기하여 하드 필터(예산·지역·방수·입주일)와 소프트 선호(채광·층·역세권·주차·반려동물 등)를 구조화
- **매물 검색** — **직방 / 다방 / 피터팬의 좋은방 구하기**를 브라우저로 탐색하고, 가능한 경우 국토부(MOLIT) 실거래가·KB시세로 비교 (Computer use 필요). ⛔ **네이버 부동산은 자동 탐색하지 않습니다**(약관·robots.txt·판례상 크롤링 금지) — 네이버 매물은 사용자가 URL을 붙여넣으면 그것만 분석
- **허위매물 가드레일** — 직거래(피터팬) 비중이 높은 만큼, 주변 시세 대비 비정상적 저가, 사진·주소 중복, "방금 계약됨" 유도, 검증 배지 부재 등을 미끼 신호로 잡아 `미끼_의심` 표시. 역이미지 검색·KISO 매물클린센터 신고 이력·등기부 대조 같은 추가 검증도 안내
- **적합도 랭킹** — 하드 필터 위반 매물 제외 후, 매물별 강·약점을 설명하며 정렬
- **매물 위험 점검** — 전세가율(깡통전세), 근생빌라, 동시진행 신축빌라, 신탁사기 등 위험 신호를 🟢/🟡/🔴 배지로 표시. **깡통전세 회수여력 공식**(`시세×경매낙찰가율 − 선순위보증금 − 융자 − 내 보증금`)과 등기부 갑구·을구(근저당 채권최고액, 임차권등기명령) 관점으로 점검
- **계약서·특약 검토** — 표준임대차계약서 사용 여부, 보증금/지급방식(임대인 본인 명의 계좌), 계약기간·**계약갱신청구권**(2+2)·임대료 5% 상한, 수리의무·전세대출·보증보험 **특약** 문구를 짚어 누락·불리 조항을 점검
- **보증금 보호 설계** — 전입신고+확정일자(대항력·우선변제권)와 잔금일 절차, **전세보증금 반환보증 HUG/HF/SGI 비교**, 보증금 미반환 시 대응 절차(내용증명→임차권등기명령→보증이행/지급명령→소송) 안내
- **중개사 문의** — 한국어 문의 메시지 **초안을 작성**해 복사용 코드 블록과 **"메시지 앱에서 열기" 버튼**(`sms:` 딥링크)으로 제시. 발송은 사용자가 직접. 🔴 매물은 위험 경고 확인을 추가로 요구
- **입주 전 집 상태 체크리스트** — 수도·전기·보일러·채광·보안 등 viewing 시 확인 항목 제공
- **계약 전 실사 체크리스트** — 서명 전 확인해야 할 서류(등기부·건축물대장·세금 등) 목록 제공

## 설치 방법 (Claude 데스크톱 앱)

이 스킬은 **Claude 데스크톱 앱**에 zip 파일로 업로드하여 사용합니다.
커스텀 스킬은 **Pro · Max · Team · Enterprise 요금제**에서 코드 실행이 켜져 있어야 사용할 수 있습니다.

### 1단계 — 코드 실행 기능 켜기

`설정(Settings) → Capabilities`에서 **Code execution and file creation(코드 실행 및 파일 생성)**을 켭니다.

### 2단계 — 스킬 zip 내려받기

GitHub 저장소에서 zip을 받습니다.
```bash
git clone git@github.com:hth225/find-property.git
cd find-property
# 폴더 자체가 루트에 들어가도록 압축 (SKILL.md만 담으면 안 됩니다)
zip -r find-property.zip SKILL.md reference
```
> 또는 GitHub의 **Code → Download ZIP** 버튼으로 받아도 됩니다.
> 핵심은 zip 안에 `SKILL.md`와 `reference/` 폴더가 **함께** 들어 있어야 한다는 점입니다.

### 3단계 — 데스크톱 앱에 업로드

`설정(Settings) → Customize → Skills → Upload`에서 위 zip 파일을 업로드합니다.
업로드하면 Claude가 `SKILL.md`를 읽어 스킬 이름·설명을 자동으로 표시합니다.

### 4단계 — 활성화 확인

스킬 목록에 **find-property**가 나타나면, 토글을 켜서 활성화합니다.
끄거나 삭제하려면 스킬 옆 `···` 메뉴를 사용하세요.

### (중요) 브라우저 검색을 위한 Claude Chrome 확장 프로그램

이 스킬의 **SEARCH 단계**는 직방·다방·피터팬을 **실제 브라우저로 탐색**합니다(네이버 부동산은 자동 탐색 제외).
데스크톱 앱에서 이 기능을 쓰려면 **Claude for Chrome(크롬 확장 프로그램)**을 설치·연결하세요.

- Claude in Chrome은 **베타**이며 **유료 요금제(Pro·Max·Team·Enterprise)**에서 Google Chrome / Microsoft Edge로 사용할 수 있습니다.
- 확장 프로그램이 Claude에게 웹페이지를 읽고·클릭하고·이동하는 권한을 부여해, 실매물 검색이 가능해집니다.
- 확장 프로그램 없이도 스킬은 동작하지만, 이 경우 Claude가 **매물 URL을 직접 붙여넣어 달라고 요청**한 뒤 그 매물만 정규화·분석합니다.

> Claude Code 사용자라면 저장소를 `~/.claude/skills/find-property`로 clone 하면 됩니다. 단, 본 README는 데스크톱 앱 기준입니다.

## 사용 방법

설치 후 Claude에게 한국에서 집을 찾고 싶다고 말하면 스킬이 활성화됩니다.

예시 — 처음부터 찾기:
> "서울 마포구에서 전세 2억 5천 이하, 방 2개 빌라 찾아줘"

> "월세 매물 후보 추려주고 중개사한테 집 보러 가고 싶다고 문의 보내줘"

예시 — 매물 위험 검토만:
> "직방에서 본 이 매물 전세사기 위험한지 봐줘: [URL]"

> "시세 4억, 선순위 근저당 1억, 보증금 3억 5천인데 깡통전세야?"

예시 — 계약서·특약 검토만:
> "받은 임대차계약서 사진인데 빠진 특약이나 불리한 조항 있는지 봐줘"

> "이 집 전세보증보험 들 수 있어? HUG랑 SGI 중 뭐가 나아?"

> "전입신고랑 확정일자 언제 해야 안전해?"

그러면 Claude가 다음 순서로 진행합니다(특정 단계만 요청하면 해당 단계만 수행).

1. **INTERVIEW** — 조건을 한 번에 하나씩 질문하여 검색 기준(`criteria`)을 정리
2. **SEARCH** — 직방/다방/피터팬에서 후보 매물을 수집하고 허위매물 신호를 걸러냄 (네이버 부동산은 자동 탐색 제외 — URL 붙여넣기로만 분석. 브라우저 사용 불가 시에도 매물 URL을 붙여넣어 달라고 요청)
3. **RANK** — 하드 필터로 거르고, 전세/월세는 사기 위험 배지를 붙여 정렬한 숏리스트 제시
4. **CONTACT** — 선택한 매물의 문의 메시지 **초안**을 복사용 블록 + **"📩 메시지 앱에서 열기" 버튼**으로 제시 → **사용자가 직접 발송** (🔴 매물은 위험 경고 확인 필요)
5. **DUE-DILIGENCE** — 계약 전 확인할 서류 체크리스트 핸드오프

### 알아두면 좋은 점

- **Claude for Chrome(크롬 확장 프로그램)**이 연결돼 있어야 실매물 검색이 가능합니다. 없으면 매물 URL을 직접 제공하면 됩니다.
- Claude는 **메시지를 직접 발송하지 않습니다.** 문의 초안을 만들어 복사·"메시지 앱에서 열기" 버튼으로 건네주면, **발송은 사용자가 직접** 합니다. 보증금 이체·계약도 하지 않으며, 목적은 집 보러 가는 약속(viewing) 잡기까지입니다.
- 전세가율 등 산정이 불가능한 값은 추측하지 않고 "산정 불가"로 표기하며, 불투명함 자체를 위험으로 취급합니다.
