한국 부동산(전세·월세·매매)을 찾고 평가한 뒤 중개사에게 연락까지 도와주는 **Claude 스킬(Skill)**입니다.

니즈 인터뷰 → 매물 검색 → 적합도 랭킹 → 전세사기 위험 점검 → 중개사 문의 작성/발송 → 계약 전 실사 핸드오프까지, 집 구하는 전 과정을 단계별로 안내합니다.

> ⚠️ 본 스킬은 참고용이며 법률·중개 자문이 아닙니다. 계약 전 반드시 등기부등본·건축물대장 확인과 공인중개사·HUG·법률 전문가 상담을 거치세요.

---

## 주요 기능

- **적응형 니즈 인터뷰** — 거래유형(전세/월세/매매)과 건물유형에 따라 질문을 분기하여 하드 필터(예산·지역·방수·입주일)와 소프트 선호(채광·층·역세권·주차·반려동물 등)를 구조화
- **매물 검색** — 네이버 부동산 / 직방 / 다방을 브라우저로 탐색하고, 가능한 경우 국토부(MOLIT) 실거래가 비교 (Computer use 필요)
- **적합도 랭킹** — 하드 필터 위반 매물 제외 후, 매물별 강·약점을 설명하며 정렬
- **전세사기 위험 점검** — 전세가율(깡통전세), 근생빌라, 동시진행 신축빌라, 신탁사기 등 위험 신호를 🟢/🟡/🔴 배지로 표시
- **중개사 문의** — 한국어 문의 메시지 초안 작성 후, **사용자 승인 시에만** 발송. 🔴 매물은 위험 경고 확인을 추가로 요구
- **계약 전 실사 체크리스트** — 서명 전 확인해야 할 서류 목록 제공

## 설치 방법

이 스킬은 Claude의 스킬 디렉터리에 넣으면 자동으로 인식됩니다.

### 방법 1 — git clone (권장)

**Claude Code (개인 스킬)**
```bash
git clone git@github.com:hth225/find-property.git ~/.claude/skills/find-property
```

**Claude Code (프로젝트 스킬)** — 특정 프로젝트에서만 사용
```bash
git clone git@github.com:hth225/find-property.git ./.claude/skills/find-property
```

### 방법 2 — 직접 복사

이미 zip 등으로 파일을 받았다면, `SKILL.md`와 `reference/` 폴더를 통째로 스킬 디렉터리에 넣습니다.
```bash
mkdir -p ~/.claude/skills/find-property
cp -R SKILL.md reference ~/.claude/skills/find-property/
```

### 설치 확인

스킬 디렉터리에 `SKILL.md`가 위치하면 됩니다. Claude Code에서는 `/help`나 스킬 목록에서 `find-property`가 보이는지 확인하세요.
```
~/.claude/skills/find-property/SKILL.md   ✅
```

> Claude Code 외 환경(Claude.ai, API 등)에서는 각 플랫폼의 스킬 등록 방식을 따르세요. 핵심은 `SKILL.md`와 `reference/` 폴더가 한 디렉터리에 함께 있어야 한다는 점입니다.

## 사용 방법

설치 후 Claude에게 한국에서 집을 찾고 싶다고 말하면 스킬이 활성화됩니다.

예시:
> "서울 마포구에서 전세 2억 5천 이하, 방 2개 빌라 찾아줘"

> "직방에서 본 이 매물 전세사기 위험한지 봐줘: [URL]"

> "월세 매물 후보 추려주고 중개사한테 집 보러 가고 싶다고 문의 보내줘"

그러면 Claude가 다음 순서로 진행합니다.

1. **INTERVIEW** — 조건을 한 번에 하나씩 질문하여 검색 기준(`criteria`)을 정리
2. **SEARCH** — 네이버 부동산/직방/다방에서 후보 매물을 수집 (브라우저 사용 불가 시, 매물 URL을 붙여넣어 달라고 요청)
3. **RANK** — 하드 필터로 거르고, 전세/월세는 사기 위험 배지를 붙여 정렬한 숏리스트 제시
4. **CONTACT** — 선택한 매물의 문의 메시지를 보여주고 **승인 후** 발송 (🔴 매물은 위험 경고 확인 필요)
5. **DUE-DILIGENCE** — 계약 전 확인할 서류 체크리스트 핸드오프

### 알아두면 좋은 점

- **Computer use(브라우저)가 켜져 있어야** 실매물 검색이 가능합니다. 비활성 상태라면 매물 URL을 직접 제공하면 됩니다.
- Claude는 **사용자 승인 없이 메시지를 보내거나 보증금을 이체하거나 계약하지 않습니다.** 목적은 집 보러 가는 약속(viewing) 잡기까지입니다.
- 전세가율 등 산정이 불가능한 값은 추측하지 않고 "산정 불가"로 표기하며, 불투명함 자체를 위험으로 취급합니다.
