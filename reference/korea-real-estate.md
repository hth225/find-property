# 한국 부동산 참조 자료

이 문서는 `find-property` 스킬이 INTERVIEW · RANK · CONTACT 단계에서 로드하는 참조 자료입니다.
법적·중개 조언이 아니며, 사용자가 직접 판단하여 활용해야 합니다.

## 용어 사전 (Glossary)

**전세** — 임차인이 집주인에게 목돈(보증금)을 맡기고, 계약 기간(통상 2년) 동안 월세 없이 거주한 뒤 계약 종료 시 보증금 전액을 돌려받는 임대 방식. 한국 고유의 제도로, 임대인은 그 보증금을 굴려 수익을 얻는다.

**월세** — 매월 일정 금액을 임대료로 지급하는 방식. 소액 보증금(수백만~수천만 원)과 함께 계약하는 경우가 일반적이다.

**매매** — 부동산의 소유권 자체를 사고파는 거래. 계약금(통상 10%) → 중도금(선택) → 잔금 지급과 동시에 소유권이전등기를 마치는 순서로 진행된다.

**보증금** — 전세 또는 월세 계약 시 임차인이 임대인에게 맡기는 예치금. 계약 종료 후 반환이 원칙이며, 반환 리스크가 핵심 점검 대상이다.

**전세가율** — (전세보증금 ÷ 매매가격) × 100. 통상 70~80%를 초과하면 '깡통전세' 위험 구간으로 분류된다. 전세가율이 높을수록 집값 하락 시 보증금 미반환 위험이 커진다.

**근저당권** — 채권자(주로 은행)가 담보로 설정하는 저당권. 등기부등본 을구(乙區)에 기재되며, 채무불이행 시 경매로 배당된다. 선순위 근저당 채권액이 크면 임차인 보증금 회수가 어려워진다.

**등기부등본** — 부동산 권리관계를 공시하는 공적 장부. 표제부(소재지·면적), 갑구(소유권·가압류·가등기), 을구(근저당·전세권 등 제한물권)로 구성된다. 대법원 인터넷등기소(iros.go.kr)에서 열람·발급 가능하다.

**건축물대장** — 건물의 구조·용도·면적·위반 건축 등을 기재한 공적 장부. 정부24(gov.kr) 또는 세움터(eais.go.kr)에서 발급한다. 실제 용도와 등재 용도가 다르면 전입신고 불가 및 대출 제한 등의 문제가 생길 수 있다.

**확정일자** — 임대차 계약서에 공증 효력을 부여하는 날짜 도장. 주민센터 또는 온라인(인터넷등기소)에서 받을 수 있다. 경매 배당 시 우선변제권의 기준 시점이 된다.

**전입신고** — 새 주소지 주민센터에 거주지 변경을 신고하는 절차. 전입신고 + 확정일자를 갖추면 대항력과 우선변제권이 생긴다.

**대항력** — 임차인이 제3자(새 집주인, 낙찰자 등)에게 임대차 관계를 주장할 수 있는 권리. 주택 인도(이사) + 전입신고를 마친 다음 날 0시부터 발생한다. 대항력이 없으면 건물이 경매로 넘어갔을 때 임차인이 퇴거해야 할 수 있다.

**관리비** — 공동주택 유지·운영에 드는 비용을 입주자가 분담하는 금액. 항목별 내역이 관리비 고지서에 명시돼야 하며, 인터넷·TV·수도·전기·청소·엘리베이터·택배함 등이 포함되기도 한다. 관리비 섹션에서 상세 설명.

**전용면적** — 해당 세대가 단독으로 사용하는 내부 면적. 법적 계약·과세 기준 면적.

**공급면적** — 전용면적 + 공용면적(계단·복도·엘리베이터 등의 지분)을 합산한 면적. 분양·광고에서 자주 표기되어 실제보다 크게 느껴진다.

**오피스텔** — 업무용으로 허가된 집합건물이지만 주거 용도로도 사용되는 형태. 주거용과 업무용 간 세금·대출 조건이 다르며, 전용면적 계산 방식이 아파트와 상이하다.

**빌라 (다세대/연립)** — 다세대주택(연면적 660㎡ 이하, 4층 이하, 각 세대가 구분 소유)과 연립주택(연면적 660㎡ 초과, 4층 이하)의 통칭. 아파트보다 등기·관리가 복잡하고 실거래가 데이터가 희소하여 감정이 어렵다.

**원룸** — 주방·욕실을 제외한 공간이 하나의 방으로 구성된 소형 주거 형태. 법적 정의가 없어 매물마다 구조 차이가 크다.

**다가구주택** — 단독주택으로 등기되지만 내부가 여러 세대로 나뉘어 임대되는 건물(3층 이하, 연면적 660㎡ 이하). 소유자(집주인)는 1인이며, 각 세대는 구분 소유권이 없다. 선순위 임차인 보증금 합계를 확인해야 한다.

**근린생활시설 (근생)** — 상업 용도로 허가된 건물 1층·저층부 공간. 근생 건물의 위층을 주거 용도로 전용(違用)하여 임대하는 경우가 많으며, 이 경우 전입신고가 불가능하여 대항력·우선변제권을 취득할 수 없다. 건축물대장에서 반드시 확인해야 한다.

**신탁** — 부동산 소유자(위탁자)가 신탁회사(수탁자)에 소유권을 이전하여 관리·처분을 맡기는 법률 구조. 신탁 등기된 물건을 임차할 때는 수탁자(신탁회사)의 동의 없이 체결된 계약은 대항력이 없을 수 있다. 을구에 '신탁' 표기가 있으면 반드시 신탁원부를 확인한다.

**깡통전세** — 전세보증금이 매매가에 근접하거나 초과하여, 집값 하락이나 경매 시 임차인이 보증금을 전액 회수하지 못할 위험이 있는 상태. 전세가율 70% 이상 구간에서 위험이 급증하며, 빌라·오피스텔에서 특히 빈번하다.

**환산보증금** — 월세 계약에서 보증금을 전세 기준으로 환산한 금액. 계산식: `환산보증금 = 보증금 + (월세 × 100)`. 중개수수료 요율 산정 기준으로도 사용된다.

## 매물 사이트 & 검색 패턴

### 주요 플랫폼

**네이버 부동산 (land.naver.com)**
한국 최대 규모의 부동산 매물 포털. 매물 수가 가장 많고 지도·단지·실거래가를 통합 제공한다.

**직방 (zigbang.com)**
아파트·오피스텔·빌라·원룸에 특화된 앱 중심 플랫폼. 3D 단지 뷰, 등기 정보 연동 기능을 제공한다.

**다방 (dabangapp.com)**
원룸·투룸·오피스텔 등 소형 주거 매물에 강점. 방 구조·옵션 필터가 세분화되어 있다.

### 기술적 제약 — JS 렌더링 + 안티봇

세 플랫폼 모두 **JavaScript로 동적 렌더링**되며, 크롤링 방지 조치(Cloudflare, 자체 봇 탐지)가 적용되어 있다.
- `plain fetch` / `curl` 로는 실제 매물 데이터를 가져올 수 없다.
- 반드시 **브라우저 세션** (Computer Use 또는 Playwright 기반 자동화)을 통해 접근해야 한다.
- 무분별한 자동화는 각 사이트 이용약관 위반이 될 수 있다 — ToS 섹션 참조.

### 브라우저 세션에서 설정해야 할 검색 패싯

| 패싯 | 설명 |
|------|------|
| 지역 | 시/구/동 또는 단지명 |
| 거래유형 | 매매 / 전세 / 월세 |
| 가격범위 | 최소~최대 보증금 또는 매매가 |
| 면적 | 전용면적(㎡) 또는 평수 범위 |
| 방 수 | 방 개수 (원룸/1룸/2룸/3룸 이상) |
| 건물유형 | 아파트 / 오피스텔 / 빌라(다세대·연립) / 단독·다가구 / 원룸 |

### MOLIT 실거래가 공개시스템

국토교통부 실거래가 공개시스템(rt.molit.go.kr) 및 공공데이터포털 OpenAPI를 통해 실제 거래된 가격 데이터를 조회할 수 있다.

**제약사항:**
- **서비스 키(API Key)** 필요 — 공공데이터포털(data.go.kr)에서 신청.
- 데이터는 **지역 단위 집계** 수준으로 제공되며, 특정 동호수 거래 내역은 노출되지 않는 경우가 많다.
- 빌라·오피스텔은 아파트에 비해 **거래 건수 자체가 희소**하여 비교 기준 삼기 어렵다.
- 신고 지연(통상 계약 후 30일 이내 신고 의무)으로 인해 최신 거래가 빠질 수 있다.

## 관리비 기준 (norms)

### 일반적 관리비 포함 항목

공동주택 관리비 고지서에는 다음 항목이 분리 고지되어야 한다:

| 항목 | 설명 |
|------|------|
| 인터넷 | 단지/건물 공용 인터넷 회선 분담금 |
| TV (유선방송) | 케이블TV·위성 공용 계약 분담금 |
| 수도 | 상수도 사용료 분담 |
| 전기 | 공용 전기(복도·엘리베이터·로비 등) 분담 — 세대별 전기는 별도 고지 |
| 청소 | 복도·계단·주차장 등 공용 공간 청소 비용 |
| 엘리베이터 | 유지보수·검사 비용 |
| 택배함 | 무인택배함 유지·운영 비용 |
| 주차 | 주차장 운영 비용 (별도 청구인 경우도 있음) |
| 장기수선충당금 | 대형 수선 대비 적립금 (아파트·오피스텔 해당) |

관리비 항목이 한 줄로 뭉뚱그려져 있거나 금액이 지역 시세 대비 현저히 높으면, **실질 월세를 낮게 포장하는 수단**으로 사용되는 사례가 있다.

> 사기 주의: 공식 월세는 낮게 설정하고 관리비를 부풀려 실질 임대료를 은폐하는 수법 — 사기 상세 내용은 `reference/jeonse-fraud-check.md`의 월세 섹션을 참조.

## 매매 due-diligence

매매 계약 전 아래 항목을 순서대로 점검한다.

### 1. 등기부등본 확인

계약 직전(당일)에 대법원 인터넷등기소에서 열람하여 최신 권리관계를 확인한다.

- **갑구**: 가압류·가처분·가등기·예고등기 여부 — 소유권 분쟁 위험
- **을구**: 근저당권 설정 금액 합계 — 매매가 대비 채권최고액 비율 검토

### 2. 근저당 말소 조건 특약

잔금 지급 전 근저당권이 말소되어야 한다면, 특약 문구를 계약서에 명시해야 한다.
예시 특약: `"잔금 지급일까지 을구에 기재된 모든 근저당권을 말소하기로 한다. 미이행 시 매수인은 계약을 해제하고 계약금의 배액을 청구할 수 있다."`

### 3. 잔금 · 소유권이전 동시이행

잔금 지급과 소유권이전등기 신청은 **같은 날 동시에** 이루어져야 한다. 법무사를 통해 잔금 수령 직후 등기를 신청하고, 당일 등기 완료를 확인한다.

### 4. 가등기 · 가압류 확인

가등기(소유권 이전 예약)와 가압류는 매수 후에도 승계될 수 있다. 원칙적으로 잔금 지급 전 말소를 요구하거나, 말소를 조건으로 한 특약을 반드시 삽입한다.

### 5. 실거래가 대비 가격 검토

MOLIT 실거래가 공개시스템에서 동일 단지·동일 면적·동일 층의 최근 거래가를 조회하여, 매도 호가가 시세 대비 합리적인지 판단한다. 빌라·오피스텔은 비교 사례가 부족할 수 있으므로 인근 매물 여러 건을 교차 확인한다.

### 6. 재건축 · 재개발 권리관계

재건축·재개발 구역으로 지정된 물건은 조합원 자격 요건, 분담금 예정액, 이주 시기 등을 반드시 확인한다.

- 조합설립인가 전후에 따라 권리 산정 기준이 다르다.
- 투기과열지구에서는 재건축 조합원 지위 양도 제한이 있을 수 있다.
- 관리처분계획 인가 후 명도 소송 위험이 있는 세입자가 있는지 파악한다.

## 중개수수료 (복비)

### 환산보증금 계산

월세 계약의 중개수수료는 전세 기준으로 환산한 금액(환산보증금)에 요율을 적용한다.

```
환산보증금 = 보증금 + (월세 × 100)
예: 보증금 1,000만원 / 월세 50만원 → 환산보증금 = 1,000 + (50 × 100) = 6,000만원
```

### 법정 상한요율표 (주택 기준, 공인중개사법 시행규칙 별표1)

요율은 거래유형에 따라 **임대차(전세·월세)** 와 **매매·교환** 이 서로 다르다.
월세는 위 환산보증금에 임대차 요율을 적용한다.

**임대차(전세·월세) 상한요율**

| 거래금액(환산보증금) 구간 | 상한 요율 | 한도액 |
|-------------|----------|------|
| 5천만원 미만 | 0.5% | 20만원 |
| 5천만원 이상 ~ 1억원 미만 | 0.4% | 30만원 |
| 1억원 이상 ~ 6억원 미만 | 0.3% | — |
| 6억원 이상 ~ 12억원 미만 | 0.4% | — |
| 12억원 이상 ~ 15억원 미만 | 0.5% | — |
| 15억원 이상 | 0.6% | — |

**매매·교환 상한요율**

## 중개사 메시지 템플릿 (KR)

아래 템플릿을 사용할 때 `{중괄호}` 부분을 직접 채운다. 개인 연락처 등 민감 정보는 기본값에 포함하지 않는다 — 직접 입력하거나 전화 통화로 전달한다.

### 문의 (Inquiry)

```
안녕하세요, {매물주소/동호} {거래유형} 매물 문의드립니다.
혹시 방문(집보기) 가능할까요? {희망시간1} 또는 {희망시간2} 중 가능하신 시간 있으신지요.
편하신 시간 알려주시면 맞추겠습니다. 감사합니다.
```

### 방문 확정 (Confirm Visit)

```
안녕하세요, 이전에 {매물주소/동호} {거래유형} 관련 문의드렸던 사람입니다.
{확정일시}에 방문하는 것으로 확정하겠습니다.
혹시 연락처나 추가 안내가 필요하시면 말씀해 주세요. {연락수단_플레이스홀더}
당일 뵙겠습니다. 감사합니다.
```

> PII 주의: `{연락수단_플레이스홀더}` 자리에 전화번호·이메일 등을 직접 기입하기 전, 해당 정보를 공유해도 되는 채널인지 확인한다.

### 발송 방식 — 메시지 초안 + "메시지 앱에서 열기" 버튼

스킬은 메시지를 **직접 발송하지 않는다.** 대신 사용자가 한 번에 보낼 수 있도록 두 가지를 제시한다.

1. **복사용 초안**: 완성된 한국어 메시지를 코드 블록(```)으로 출력 → 원클릭 복사 버튼 제공.
2. **메시지 앱에서 열기 버튼**: 중개사 전화번호로 `sms:` 딥링크를 만들어, 본문을 URL 인코딩해 미리 채운다.

```
[📩 메시지 앱에서 열기](sms:{전화번호}&body={URL인코딩된_메시지})
```

- 예: `[📩 메시지 앱에서 열기](sms:01012345678&body=%EC%95%88%EB%85%95%ED%95%98%EC%84%B8%EC%9A%94...)`
- Apple 기기에서는 `sms:` 대신 `imessage:`를 쓸 수 있다.
- 일부 OS는 구분자로 `&` 대신 `?`나 `;`를 요구할 수 있다 (`sms:01012345678?body=...`).
- 매물에 전화번호가 노출되지 않으면 버튼 대신 복사용 초안 + 매물 사이트 문의 링크를 제공한다.
- 본문에는 사용자의 개인 연락처(PII)를 기본 포함하지 않는다 — 사용자가 직접 추가하도록 둔다.

## ToS / 법적 주의

- **이용약관**: 네이버 부동산, 직방, 다방은 각각의 서비스 이용약관에서 자동화 접속(크롤링·스크래핑)을 금지하거나 제한하는 조항을 두고 있다. 브라우저 자동화를 통한 매물 조회나 메시지 발송이 해당 약관에 위반될 수 있으며, 계정 정지·법적 조치로 이어질 수 있다.

- **사용자 책임**: 이 플러그인이 제공하는 자동화 흐름(브라우저 조작, 문자 발송 등)의 실행 여부는 **사용자가 직접 판단하고 결정**한다. 플러그인 개발자 및 오케스트레이터는 이용약관 위반, 중개 분쟁, 계약 손실 등에 대한 법적 책임을 지지 않는다.

- **중개 조언 아님**: 이 문서 및 플러그인 전체는 정보 제공 목적이며, 법적 자문 또는 공인중개사의 중개 행위를 대체하지 않는다. 실제 계약 전 반드시 공인중개사 또는 법률 전문가와 상담한다.

- **개인정보**: 중개사에게 연락 시 전화번호·이메일 등 개인정보를 입력하기 전, 해당 정보가 어떻게 사용·저장되는지 확인한다.
